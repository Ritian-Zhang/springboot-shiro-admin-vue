# admin-api
项目基于 Spring Boot 2.1.1 、 appache shiro、 mybatis、redis、Vue的前后端分离的权限管理系统， 权限控制采用 RBAC（Role-Based Access Control，基于角色的访问控制），前端菜单支持动态路由
#### 后端技术栈

- 基础框架：Spring Boot 2.1.1.RELEASE
- 持久层框架：mybatis mybatis-plus
- 安全框架：shiro
- 缓存框架：Redis
- 日志打印：logback+log4j
- 接口文档 swagger2
- 其他：fastjson、aop、等

#### 前端技术栈
- node
- vue
- vue-router
- axios
- element ui
